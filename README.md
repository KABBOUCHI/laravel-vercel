## Installation

You can install the package via composer:

```bash
composer require kabbouchi/laravel-vercel
```

Publish stubs:

```bash
php artisan vendor:publish --provider="KABBOUCHI\LaravelVercel\LaravelVercelServiceProvider"
```

Deploy:

```bash
vercel deploy
```
